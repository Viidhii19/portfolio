# 🌐 Vidhi Gandhi — Portfolio Website

**Tech Stack:** HTML · Tailwind CSS · JavaScript

**🚀 Overview**

A clean, responsive portfolio website showcasing my work, skills, and achievements in AI/ML, Cybersecurity, and Full-Stack Development. Built with Tailwind CSS for modern styling and optimized for both desktop and mobile viewing.

**⚙️ Features**

- Professional personal branding with minimalist design
- Interactive project showcase with live links and descriptions
- Responsive layout optimized for all screen sizes
- SEO-friendly meta tags for better visibility
- Fast loading speed with Tailwind optimization

**🧩 Sections**

- Home: Brief introduction and tagline
- About: Education, interests, and technical overview
- Projects: Highlighted academic and personal projects
- Skills: Tools, languages, and technologies mastered
- Contact: Direct email and LinkedIn integration
